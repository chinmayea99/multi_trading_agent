\# multi\_trading\_agent — Multi-Agent LLM Financial Trading Framework



M.Tech thesis project comparing single-agent vs multi-agent LLM frameworks

for Indian stock market trading simulation.



\## Structure

\- data/         — price data, news data, events calendar

\- agents/       — analyst, researcher, trader agents

\- utils/        — shared utilities (cache, logger, rate limiter)

\- simulator/    — portfolio and backtest executor

\- evaluation/   — metrics and visualization

\- experiments/  — experiment configs and results

\- notebooks/    — exploratory notebooks
